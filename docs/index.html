1. Create a Python venv:

                    
                    python3 -m venv .venv

    The .venv is very important.

  2. Activate the venv:

    Windows (CMD):

                    
                .venv\Scripts\activate

    CMD may not be accurate

    macOS/Linux:

                    
                    source .venv/bin/activate

  3. Install Django:

                    
                    pip install django

  4. Add a superuser:

                    
                    python manage.py createsuperuser

    The terminal will guide you through the superuser creation.

  5. Start the server:

                    
                    python manage.py runserver

  6. Go to http://localhost:8000/admin (assuming you're on the same server you started Django on.)

  7. Sign in with the credentials you made with the python manage.py createsuperuser command.

  8. Once you're logged in, find E educate admins, and click it.

  9. Push CREATE E EDUCATE ADMIN

  10. In the user dropdown, choose the username of the superuser you created. (for every eEducate Admin, there must be a corresponding superuser)

  11. In the username field, enter a unique username

  12. In the password field, enter a secure password.

  13. In the Authorized apps field, you need to have the app names inside DOUBLE quotes, inside the square brackets, seperated by commas. Availible options are: LMS, SIS, Exam, and Curriculum.

  14. Then push the save button that fits your needs.
